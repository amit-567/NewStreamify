- 🔐 JWT Authentication & Protected Routes

- 🚨 Error Handling (Frontend & Backend)
- 🚀 Free Deployment
- 🎯 Built with Scalable Technologies like Stream
- ⏳ And much more!

## 🧪 .env Setup

### Backend (`/backend`)

```
PORT=5001
MONGO_URI
STEAM_API_KEY
STEAM_API_SECRET
JWT_SECRET_KEY
NODE_ENV=development
```

### Frontend (`/frontend`)

```
VITE_STREAM_API_KEY
```

---

## 🔧 Run the Backend

```bash
cd backend
npm install
npm run dev
```

## 💻 Run the Frontend

```bash
cd frontend
npm install
npm run dev
```
